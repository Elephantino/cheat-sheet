как найти %

нужно умножить число на сотую долю искомого процента

37% от числа 200 === 200 х 0,37 = 74

20% от числа 200 === 200 х 0,20 = 40

среднее значение и медиана

среднее значение -- сумма элементов деленая на их кол-во.

медиана -- среднее значение набора чисел(массива).

[1, 2, 3, 4, 5] -- среднее 3 / медиана 3.

[1, 2, 5, 4, 3] -- среднее 3 / медиана 5.

[1, 2, 50, 4, 3] -- среднее 12 / медиана 50.

найти среднее значение:

[1, 2, 50, 4, 3] -- 1 + 2 + 50 + 3 + 3 / 5 = 12(делим на 5 потому что количество элементов 5)
